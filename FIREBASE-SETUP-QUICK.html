‚ö° 3-Minute Firebase Setup

1 Create Firebase Project

Go to https://console.firebase.google.com

  • Click "Add project"
  • Enter: "CredHive X"
  • Click "Create project"
  • Wait 2-3 minutes for creation

2 Enable Authentication Methods

In Firebase Console:

  • Click "Authentication" (left sidebar)
  • Click "Get started"
  • Enable "Email/Password" ‚Üí click toggle ‚Üí Save
  • Enable "Google" ‚Üí click Add provider ‚Üí Save

3 Get & Copy Firebase Config

In Firebase Console:

  • Click ‚öôÔ∏è (Settings) icon (top-left)
  • Click "Project settings"
  • Scroll down to "Your apps"
  • Copy the firebaseConfig object

You'll see something like:

‚ùì FAQs

Q: Can I use the website without Firebase?

A: Yes! Fallback auth works perfectly. Users can sign up/login with email/password. Only Google OAuth requires Firebase setup.

Q: How long does Firebase setup take?

A: 3-5 minutes total. Most time is waiting for project creation.

Q: Is Firebase free?

A: Yes! Firebase has a generous free tier. No credit card required for testing.

Q: What if Google login shows "Pop-up blocked"?

A: Allow pop-ups in your browser settings. Or try on a different browser first.
